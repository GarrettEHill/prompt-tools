# Compact Result Template

Return to the coordinator using the shared schema:

- [`primitives/compact-result-schema/compact-result-schema.md`](../../primitives/compact-result-schema/compact-result-schema.md)

Required fields:

- work item id/title/type
- status
- one-line summary
- files/modules changed
- validation commands and outcomes
- PR URL and merge status
- close/update status
- blocker and retry count if applicable
- follow-up recommendation if applicable
